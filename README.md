## üìå Tools & Services Overview

### Terraform
Terraform is an Infrastructure as Code (IaC) tool used to automate the creation and management of cloud infrastructure using code. It enables consistent, repeatable, and version-controlled deployments.

---

### AWS IAM (Identity and Access Management)
AWS IAM is a service that manages access to AWS resources by defining who can access them and what actions they are allowed to perform. It helps enforce security best practices using users, roles, and policies.

---

### Git
Git is a distributed version control system used to track changes in source code and collaborate with teams. It maintains version history and supports efficient branching and merging.

---

### AWS CLI
AWS Command Line Interface (CLI) is a tool that allows users to manage and automate AWS services using command-line commands. It interacts directly with AWS APIs and is commonly used in scripting and automation.

---

### IAM Role
An IAM Role is a temporary AWS identity that provides permissions to trusted entities without using long-term credentials. It is widely used by AWS services and applications for secure access.
